# Evaluation Plan

## Goldstandard Set

Build a local evaluation set with at least 20 public websites:

- 3 B2B
- 3 SaaS
- 3 e-commerce
- 3 local service
- 2 portfolios
- 2 corporate
- 2 blogs or magazines
- 1 nonprofit
- 1 personal brand

For each website, store:

- Expected page types
- Known top issues
- Expected primary CTA
- Known mobile issues
- Known conversion issues
- Known design-system inconsistencies

## Metrics

- Finding precision
- Evidence validity
- Actionability
- Specificity
- Priority accuracy
- Coverage
- Report usefulness
- Client readiness

## Automated Checks

- Schema validation
- Screenshot existence checks
- Evidence reference checks
- Duplicate finding checks
- Generic recommendation filter
- Overclaim filter
- Priority calibration checks
